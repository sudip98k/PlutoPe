## My E-commerce using Node.js and Express
## I designed a simple ResetFul E-commerce api node.js project that where user can register himself first and set the products which he/she wants to buy.
## Prerequisites
    1 Node.js
    2 Git
## Tech Used
    1 mongoose
    2 Node.js
    3 Express.js
    4 postman
### install npm first
`npm install `
### start the server
`npm start`
